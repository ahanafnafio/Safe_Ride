/- User SQL Queries -\

Description- A guide for backend to implement code that communicates with a database. Replace '?' with actual input.


*** Registration Feature *** 

1. After user enters a valid email address, backend needs to verify there's not an entry in the database with that email already

Query: 
SELECT userID FROM Users
WHERE email = ?;


2. If email is unregistered, backend needs to store the input in the database

Query:
INSERT INTO Users (firstName, lastName, email, passwordHash, userRole)
VALUES (?, ?, ?, ?, ?);
